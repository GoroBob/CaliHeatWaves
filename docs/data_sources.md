# Data Sources Reference

| Data Source | Provider | URL | Access | Layer(s) |
|---|---|---|---|---|
| ERA5-Land | Copernicus Climate Data Store | https://cds.climate.copernicus.eu/ | Free | Layer 1 |
| MODIS LST MOD11A1 | NASA Earthdata | https://earthdata.nasa.gov/ | Free | Layer 1 |
| IDEAM DHIME | Instituto de Hidrología, Meteorología y Estudios Ambientales | https://www.ideam.gov.co/ | Free (with account) | Layer 1 |
| CVC Portal Hidroclimatológico | Corporación Autónoma Regional del Valle del Cauca | https://cvc.gov.co/ | By request | Layer 1 |
| DAGMA Air Quality | Departamento Administrativo de Gestión del Medio Ambiente (Cali) | https://www.cali.gov.co/dagma/ | By request | Layer 1, Layer 3 |
| DANE EEVV Microdata | Departamento Administrativo Nacional de Estadística | https://www.dane.gov.co/ | Free | Layer 2 |
| SISPRO RIPS | Sistema de Información de Precios de Medicamentos | https://sispro.minsalud.gov.co/ | By request | Layer 2 |
| Cali Comuna Shapefiles | Alcaldía de Cali Open Data | https://www.cali.gov.co/ | Free | All layers |
| SRTM DEM | NASA Shuttle Radar Topography Mission | https://earthexplorer.usgs.gov/ | Free | Shared geography |

## Notes

- **Free with account**: Requires registration but no payment.
- **By request**: Contact the provider directly or submit formal data request.
- **Layer designations**: Indicate primary use, though some datasets (e.g., DAGMA air quality) may inform multiple layers.
- **Microdata**: DANE EEVV and SISPRO RIPS are individual-level records; restricted access protocols apply.
